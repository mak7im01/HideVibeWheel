# HideVibeWheel

Аддон для PulseSync — убирает вертикальную карусель «Рекомендации моей волны» со страницы Моей волны в новом интерфейсе Яндекс Музыки и центрирует основной контент

---

## Благодарности

За идею взят аддон Minimalistic Wave Tab от [dieugh](https://github.com/dieughh/MinimalisticWaveTab)

---

## Что делает

- **Скрывает карусель** рекомендаций (жанры, артисты, настроения) по умолчанию
- **Кнопка «Настроить»** внизу по центру позволяет временно открыть карусель и вернуться к выбору волны
- **Центрирует** имя артиста, плеер и AI-карточку на освободившемся пространстве
- **Скрывает фото артиста** на треках где Яндекс показывает обложку вместо имени — вместо неё отображается текстовое имя крупным шрифтом

---

## Использование

- По умолчанию карусель **скрыта**
- Нажмите кнопку **«Настроить»** внизу страницы Моей волны чтобы открыть карусель
- Повторное нажатие скроет карусель обратно
- Состояние (открыта/закрыта) сохраняется между сессиями

---

## Настройки

**Скрыть «Моя волна обновилась»** — убирает баннер с уведомлением в правом верхнем углу страницы Моей волны. По умолчанию включено.
